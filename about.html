About

 This project was built in collaboration by Matt Raymond and Tristan Chilvers as an exploration of how publicly-available tools might act as a form of preventative medicine. The source code can be found here.

 This information is meant to help people make mindful choices and avoid areas with high infection rates, but is not meant to supersede the advice and/or mandates of local governments or health authorities.

 For further information about COVID-19, please visit the CDC, or visit this interactive map from John's Hopkins.

Basics

 This website relies heavily on javascript, and will not function without it. If the site is not loading correctly, please make sure that javascript is enabled.

 Location data is also required to find pertinent data, and if the website is unable to locate you, try enabling location services.

Danger Metric

 The danger level is a numeric representation of how dangerous we believe the current location to be. Most locations will be within a 1-5 dot range. However, anything considered extremely dangerous will be included as five red dots.

 This is calculated by the number of infected people at an epicenter divided by the distance between that epicenter and the location in question. The displayed danger metric is the largest value given for that location in relation to any epicenter.
