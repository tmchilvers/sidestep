About

 This project was built in collaboration by Matt Raymond and Tristan Chilvers as a way to help people stay safe during the ongoing covid-19 epidemic.

 This information is meant to help people make mindful choices and avoid areas with high infection rates, but is not meant to supersede the advice/mandates of local governments or health authorities.

 For further information about covid-19, please visit the CDC, or visit this interactive map from John's Hopkins

Basics

 This website relies heavily on javascript, and will not function correctly without it. If this website is not loading correctly for you, please make sure that javascript is enabled.

 Additionally, this website requires location data in order to find the places around you. If the website is unable to locate you, try enabling location services.

Safety Level

 This dropdown allows you to choose how safe you want to be when venturing out. Safe will have a lower danger cutoff than Risky, etc.

 Every time you choose a new value from this dropdown, it will clear your search results, as updating this requires an entirely new search.

Category

 This dropdown allows you to choose what sort of business/location you're searching for.

 This dropdown will also clear the search results, as it requires a new search to update the search results.

Units

 This dropdown allows you to choose metric or imperial as your display units.

 This can be updated without searching again, as all distances remain the same and are simply displayed differently.

Sorting

 This dropdown allows you to choose how you want to sort the search results. This doesn't require a reload as it simply changes the order of the results.

Search Radius

 This allows you to choose the radius that you want to search in.

Search Results

 Search results are displayed as small cards that can be opened or closed by tapping on the dark-grey header. Each result displays the location name, as well as your distance from it.

 Once opened, each will display a short bit of information about the location, as well as provide a link to the location in Google Mapsâ„¢.

Danger Metric

 The danger metric is simply a numeric representation of how dangerous we believe the current location to be.

 Currently, each danger metric is the number of infected people at an epicenter divided by the distance between that epicenter and the location in question. The displayed danger metric is the largest value given for that location in relation to any epicenter.
